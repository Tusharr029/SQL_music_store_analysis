# Music Store Data Analysis Using SQL

## Project Overview
This project analyzes an online music store database using SQL queries to solve real-world business problems.

The project is divided into Easy, Moderate level questions that help understand customer behavior, sales trends, music preferences, and business growth.

This project helped me improve my SQL querying and data analysis skills by working with a relational database.

---

## Objectives
The main objective of this project is to analyze the music store dataset and answer important business questions such as:
- Who is the best customer?
- Which city generates the highest revenue?
- What are the most popular music genres?
- Which artists produce the most rock music?
- How much money do customers spend on artists?
